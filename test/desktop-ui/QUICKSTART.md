# NemesisBot Desktop UI - Quick Start Guide

## 🚀 快速启动

### Windows

```bash
cd test/desktop-ui

# 方式 1: 使用快速启动脚本
start.bat

# 方式 2: 直接运行
desktop-ui.exe desktop
```

### Linux/macOS

```bash
cd test/desktop-ui

# 方式 1: 使用快速启动脚本
./start.sh

# 方式 2: 直接运行
./desktop-ui desktop
```

---

## 📋 可用命令

| 命令 | 说明 |
|------|------|
| `desktop-ui desktop` | 启动桌面模式（打开浏览器窗口） |
| `desktop-ui web` | 启动 Web 模式（手动打开浏览器） |

## ✨ 功能特性

### 1. 侧边栏导航
- **Chat** - 聊天界面
- **Overview** - 系统概览
- **Logs** - 日志查看
- **Settings** - 设置页面

### 2. 聊天功能
- 在输入框输入消息
- 点击 "Send" 按钮发送
- 或使用 `Ctrl+Enter` 快捷键

### 3. 主题切换
- 点击 Settings 页面
- 选择 "☀️ Light" 或 "🌙 Dark"
- 主题会自动保存

### 4. 实时状态
- 顶部显示连接状态
- 绿点 = 已连接
- 红点 = 已断开

## 🔧 技术细节

### 架构
```
desktop-ui.exe (Go 程序)
    ↓
HTTP 服务器 (内嵌)
    ↓
静态资源 (HTML/CSS/JS)
    ↓
浏览器窗口 (app 模式)
```

### 端口
- 自动分配随机端口
- 只监听本地 (127.0.0.1)

## ❓ 常见问题

### Q: 为什么使用浏览器而不是原生窗口？
A: 为了避免复杂的 webview 依赖。浏览器 app 模式可以达到类似效果。

### Q: 可以关闭浏览器窗口吗？
A: 可以，但服务器会继续运行。按 `Ctrl+C` 停止服务器。

### Q: 端口号是多少？
A: 自动分配，每次运行可能不同。程序启动时会显示。

### Q: 如何停止程序？
A: 在命令行窗口按 `Ctrl+C`。
