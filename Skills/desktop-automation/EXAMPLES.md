# 桌面自动化 Skill - 使用示例

⚠️ **重要**: 图像分析由 AI 自己完成，使用 Read 工具读取截图，不使用任何 MCP 分析功能。

---

## 示例 1: 截图分析窗口

```
用户: 帮我查看"安全审批"窗口的内容

AI:
1. 使用 window-mcp 查找窗口
2. 使用 window-mcp 截图，保存到 temp.png
3. 使用 Read 工具读取 temp.png
4. AI 分析截图内容:
   - 窗口尺寸: 600×580
   - 倒计时: 28 秒
   - 按钮: "允许"(绿色)、"拒绝"(红色)
```

---

## 示例 2: 点击按钮（视觉识别）

```
用户: 点击"安全审批"窗口的绿色"允许"按钮

AI:
1. 使用 window-mcp 查找并激活窗口
2. 使用 window-mcp 截图，保存到 temp.png
3. 使用 Read 工具读取 temp.png
4. AI 分析识别绿色按钮位置: (450, 520)
5. 计算绝对坐标并点击
6. 验证窗口关闭

结果: ✓ 成功
```

---

## 示例 3: 点击按钮（计算）

```
用户: 点击"设置"窗口的保存按钮

AI:
1. 使用 window-mcp 查找窗口
2. 获取窗口坐标: (200, 150, 800, 650)
3. 窗口尺寸: 600×500
4. 计算: 75%, 85% → (450, 425)
5. 绝对坐标: (650, 575)
6. 使用 window-mcp 点击

结果: ✓ 成功（快速）
```

---

## 关键流程

### AI 分析截图的步骤

```
1. window-mcp.capture_rect() → 保存到 file.png
2. Read(file.png) → AI 获取图像数据
3. AI 视觉分析 → 识别元素、坐标、文本等
4. 返回分析结果
```

### ❌ 错误示例

```
错误: 调用 window-mcp.analyze()
正确: window-mcp.capture_rect() → Read() → AI 分析
```
