## 📝 如何创建新任务

### 1. 创建任务文档

```bash
# 在 docs/PLAN/ 目录创建
# 文件命名规范：YYYY-MM-DD_任务名称.md
touch docs/PLAN/2026-03-24_NEW_TASK.md
```

### 2. 任务文档模板

任务文档应包含：
- **任务概述** - 为什么要做
- **任务类型** - 开发/测试/优化/文档
- **优先级** - P0/P1/P2/P3
- **预计工期** - 时间估算
- **实施步骤** - 详细任务清单
- **验收标准** - 完成条件

### 3. 更新本 README

- 将新任务添加到"所有任务概览"
- 设置正确的优先级和工期
