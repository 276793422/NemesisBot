## 📊 审批中心

### 统计卡片

页面顶部显示 5 个统计卡片：
- **总请求** - 所有审批请求总数
- **已批准** - 批准的请求数
- **已拒绝** - 拒绝的请求数
- **超时** - 超时的请求数
- **平均耗时** - 平均决策时间（秒）

### 模拟审批请求

点击不同风险等级的按钮来模拟审批请求：

| 按钮 | 操作 | 风险等级 |
|------|------|----------|
| 🟢 文件删除 | 删除指定文件 | HIGH |
| 🔴 进程执行 | 执行系统命令 | CRITICAL |
| 🟡 注册表修改 | 修改注册表 | MEDIUM |
| 🟡 网络下载 | 下载文件 | MEDIUM |
| 🔴 系统关闭 | 关闭系统 | CRITICAL |

### 审批流程

1. 点击任意模拟按钮
2. 审批对话框弹出
3. 选择 **"✓ 允许执行"** 或 **"✗ 拒绝操作"**
4. 等待倒计时结束（自动拒绝）

### 审批历史

查看最近的审批记录：
- 请求 ID
- 操作名称
- 目标文件/路径
- 决策结果（已批准/已拒绝/超时）
- 风险等级
- 响应时间

## 💬 Chat 聊天

### 发送消息

1. 在底部的输入框中输入消息
2. 点击 **"发送"** 按钮或按 `Enter` 键
3. 等待 AI 响应

### 快捷键

- `Ctrl + K` - 聚焦到输入框
- `Enter` - 发送消息
- `Ctrl + H` - 清空聊天历史

### 消息显示

- **蓝色气泡**（右侧）- 你发送的消息
- **深色气泡**（左侧）- AI 的回复
- 每条消息显示时间戳

## 📋 日志查看

### 日志级别

日志按级别分类：

| 级别 | 颜色 | 说明 |
|------|------|------|
| DEBUG | 🟦 灰色 | 调试信息 |
| INFO | 🟩 蓝色 | 一般信息 |
| WARN | 🟨 黄色 | 警告信息 |
| ERROR | 🟥 红色 | 错误信息 |

### 过滤功能

**按级别过滤**:
- 选择 "所有级别" 查看全部日志
- 选择特定级别查看该级别的日志

**按模块过滤**:
- 选择 "所有模块" 查看全部模块
- 选择特定模块（app、webview、server、agent、security等）

### 日志格式

```
[时间] [级别] [模块] 消息内容
```

示例：
```
2026-03-24 12:34:56 INFO [app] NemesisBot UI started
```

## ⚙️ 系统设置

### 系统状态

显示实时系统信息：
- **运行时间** - 应用运行时长
- **内存使用** - 当前内存占用（MB）
- **CPU 使用** - CPU 使用率百分比
- **线程数** - 活跃线程数
- **版本** - 应用版本号
- **Go 版本** - Go 运行时版本

### 配置项

| 配置项 | 类型 | 说明 |
|--------|------|------|
| bot.name | 文本 | Bot 名称 |
| security.enabled | 布尔 | 启用安全中间件 |
| security.restrict_to_workspace | 布尔 | 限制文件访问到工作区 |
| log.level | 枚举 | 日志级别 |
| cluster.enabled | 布尔 | 启用集群模式 |
| ui.theme | 枚举 | UI 主题 |
| ui.auto_theme | 布尔 | 自动切换主题 |

### 主题设置

**选择主题**:
- 🌙 **深色主题** - 默认主题，适合夜间使用
- ☀️ **浅色主题** - 明亮主题，适合日间使用

**自动切换**:
- 勾选 "自动切换主题" 可根据系统时间自动切换

## ❓ 常见问题

### Q: 如何重置聊天历史？

**A**: 在 Chat 页面点击 "清空历史" 按钮，或使用快捷键 `Ctrl + H`。

### Q: 审批请求超时时间是多少？

**A**: 默认超时时间为 30 秒。超时后自动拒绝请求。

### Q: 如何更改日志级别过滤？

**A**: 在 Logs 页面顶部，使用下拉菜单选择要查看的日志级别。

### Q: 主题设置会保存吗？

**A**: 目前主题设置在应用重启后会重置。未来的版本将支持持久化存储。

### Q: 如何退出应用？

**A**: 使用快捷键 `Ctrl + Q`，或直接关闭窗口。

### Q: 应用占用多少内存？

**A**: 应用大约占用 35-50 MB 内存，取决于使用情况。

### Q: 支持哪些操作系统？

**A**:
- ✅ Windows 10/11 (完全支持)
- 🟡 macOS (架构支持，待测试)
- 🟡 Linux (架构支持，待测试)
