# NemesisBot UI v1.0.0 发布说明

**发布日期**: 2026-03-24
**版本**: 1.0.0
**状态**: ✅ 正式发布

---

## 🎉 重大发布

NemesisBot UI v1.0.0 是从 webview2 迁移到 Wails v2 的首个正式版本。这是一个完整的桌面应用重写，提供了跨平台支持、现代化的 UI 设计和显著提升的性能。

---

## ✨ 新功能

### 1. 审批中心 📊

- ✅ 实时统计数据展示
- ✅ 5 种模拟审批请求类型
- ✅ 审批历史记录查看（最多 20 条）
- ✅ 风险等级可视化（LOW/MEDIUM/HIGH/CRITICAL）
- ✅ 事件驱动的审批流程
- ✅ 自动超时处理（30 秒）

### 2. Chat 聊天 💬

- ✅ 现代化聊天界面
- ✅ 消息历史管理
- ✅ 实时消息发送和接收
- ✅ 用户/助手消息区分显示
- ✅ 时间戳显示
- ✅ 聊天历史清空功能

### 3. Logs 日志 📋

- ✅ 实时日志查看
- ✅ 按级别过滤（DEBUG/INFO/WARN/ERROR）
- ✅ 按模块过滤（app/webview/server/agent/security等）
- ✅ 颜色编码的日志级别
- ✅ 时间戳格式化显示
- ✅ 等宽字体优化

### 4. Settings 设置 ⚙️

- ✅ 系统状态监控
  - 运行时间
  - 内存使用
  - CPU 使用率
  - 线程数
  - 版本信息
- ✅ 配置项管理
- ✅ 主题切换（深色/浅色）
- ✅ 自动主题切换
- ✅ 键盘快捷键查看

### 5. 主题系统 🎨

- ✅ 深色主题（默认）
- ✅ 浅色主题
- ✅ 一键主题切换
- ✅ 自动主题切换选项
- ✅ 平滑过渡动画

### 6. 键盘快捷键 ⌨️

- ✅ `Ctrl+1-4` - 页面快速切换
- ✅ `Ctrl+K` - 聚焦聊天输入框
- ✅ `Ctrl+H` - 清空聊天历史
- ✅ `Ctrl+L` - 清空日志
- ✅ `Escape` - 关闭对话框
- ✅ `Ctrl+Q` - 退出应用

## 📦 安装

### 方式 1: 直接运行（推荐）

```bash
# 下载 nemesisbot-ui.exe
# 双击运行
```

### 方式 2: 从源码构建

```bash
# 克隆仓库
git clone https://github.com/276793422/NemesisBot.git
cd NemesisBot/nemesisbot/wails-ui

# 构建
wails build

# 运行
./build/bin/nemesisbot-ui.exe
```

## 🔄 从旧版本迁移

### webview2 → Wails UI

1. **安装新版本**
   - 下载 `nemesisbot-ui.exe`
   - 替换旧的执行文件

2. **配置迁移**
